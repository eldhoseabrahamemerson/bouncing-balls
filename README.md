# 🏀 MDN Bouncing Balls Challenge

An interactive JavaScript animation built as part of the **MDN Web Docs – Adding Features to the Bouncing Balls Demo** challenge. This project extends the original bouncing balls animation by introducing object-oriented programming concepts, user interaction, and a simple game mechanic.

## 📖 About the Project

This project is based on the MDN Web Docs JavaScript Objects tutorial. The original bouncing balls animation has been enhanced with several interactive features, transforming it into a small game where the player controls an **Evil Circle** to capture bouncing balls.

The project demonstrates:

* JavaScript Classes
* Inheritance using `extends`
* Object-Oriented Programming (OOP)
* Collision Detection
* Canvas Animation
* Keyboard Event Handling
* DOM Manipulation

## ✨ Features

* 🎨 Colorful bouncing balls with collision effects
* ⚪ User-controlled Evil Circle
* ⌨️ Move using **W, A, S, D** keys
* 🏆 Ball counter that updates in real time
* 👾 Balls disappear when captured
* 🔄 Restart button
* 🌈 Random ball colors
* ⚡ Smooth canvas animation
